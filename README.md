
## Start Machine Learning project.

### Software and account Requirement.

Creating conda environment
```
conda create -p venv python==3.7 -y
```
```
conda activate venv/
```
OR 
```
conda activate venv
```

```
pip install -r requirements.txt
```
OR
```
python setup.py install 

```

To Add files to git
```
git add .
```

OR
```
git add <file_name>
```
